HTML
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML is used to create the structure/Framework> of a webpage
  • Common element types are METADATA elements, content division elements & semantic elements.
  • Style element defines CSS styles associated with a website/li>
  • Base element defines the base URL for a webpage
  • meta element defines metadata such as the character set, description, keywords, author & viewport.
  • Script element embeds client-side scripts in the HTML document.
  • Charset element sets character encoding
  • http-equiv element sets an HTTP header for content
  • Viewport element sets the default view for the webpage

Git

  • git branch ; list your branches and an a* will appear next to current branch
  • git status checks what branch we're on
  • git checkout -b "branch-name": creates a NEW branch and switches to it
  • git stash ; saves modified and staged changes
  • git init ; intializes an existing directory as a Git respository
  • git clones "URL" ; retrieve an entire respository from a hosted location via URL
  • git merge "branch" merge specified branch into current one
  • git commit -m "describing commit" ; commit staged content as a new commit snapshot
  • git diff ; diff of what IS CHANGED but NOT STAGED
  • git diff --staged ; diff of what is staged but NOT YET COMMITTED
  • git log ; shows the commit history for the CURRENT active branch
  • git add -A ; to commit all files in the WORKING branch "to stage"
